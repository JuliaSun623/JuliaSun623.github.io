本篇主要介绍机器学习主要概念。

Learning Map

**机器学习就是自动找函数。**例如，语音识别要找到从声音讯号到文字的函数，图像识别要找从图片到文字的函数，很会下围棋的Alpha go要找棋盘上当前的布局（黑子白子的位置）到下一步应该落子的位置的函数。

在找函数之前，首先要问以下几个问题：

  1. 你要找什么样的函数？
  2. 如何告诉机器你要找的函数？
  3. 机器如何找出对应函数？

1. 找什么样的函数

  • Regression（回归）：输出是一个标量值。「作业的Regression」
  • Classification（分类）：
    • Binary Classification：输出两个离散值（yes/no）中的一个。「作业的Classification和RNN」
    • Multi-class Classification：输出N个离散值的其中一个。「作业的CNN」
  • Generation（生成）：产生有结构的复杂信息，可以理解为让机器学习如何创造。「作业的Seq2Seq和GAN」

2. 如何告诉机器你要找的函数

  • Supervised Learning（监督学习）：给机器训练集，给出大量的输入，并且同时给出每个输入对应的正确输出。「作业的Regression、Classification、RNN、CNN和Seq2Seq」
    • 对资料作label，即要找的函数的理想输出是什么。
    • 给出label后，机器会根据给出的loss评估函数的好坏。
  • Reinforcement Learning（强化学习）：之前热度很高的Alpha go使用的技术就是强化学习。下面比较分别使用监督学习和强化学习下围棋的方法：「作业中的Reinforcement Learning」
    • 监督学习：需要告诉机器每种输入的棋盘布局，最应该下在哪里
    • 强化学习：让机器自己下围棋，告诉他结果是赢还是输（reward），如果赢，机器知道这一局下的好，如果输，机器知道这一局有一些步是不好的，而机器需要自己找出哪些不好。
  • Unsupervised Learning（无监督学习）：给机器大量数据，但是无label。「作业中的Unsupervised Learning和GAN」

3. 机器如何找到函数

  1. 给定函数寻找范围：Linear（线性函数）、Network Architecture（网络架构：RNN，CNN）
  2. 函数寻找方法：Gradient Descent（梯度下降）。「前两个作业是自行实现，之后的作业都实用pytorch中的内置函数」

一些前沿技术

Adversarial Attack

人用恶意地目的，特意制造一些noise，来判断机器的鲁棒性。

Network Compression

得到很高的准确率有可能需要一个很大的模型，这个任务的目的就是想压缩模型，使其能移植到手机等移动设备上。

Anomaly Detection

机器处理结果之外的数据。例如动物识别系统，如果输入的是一个冰箱，那机器需要知道自己无法判断。

Transfer Learning (Domain Adversarial Learning)

一般的机器学习会分为训练集和测试集，而这两个数据集的相似度一般比较高，是一个类型的数据。例如训练黑白手写数字，测试集也是黑白手写数字，但是现实中测试集有可能是彩色的手写数字，这时原本的方法就不适用。

Meta Learning

机器学习是人类设计学习方法，机器具备学习的能力；而元学习则是机器具备学习如何学习的能力。

Life-long Learning

终身学习，即可以不断地学习，学习新知识会结合旧的知识。
